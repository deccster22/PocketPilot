# When Not to Force Action

## What this means
Sometimes the smartest move is not more speed or more conviction.

## Why it matters
If the market has weak [strategy fit](../glossary/strategy-fit.md), poor [liquidity](../glossary/liquidity.md), or only thin [estimated confidence](../glossary/estimated-confidence.md), forcing action can create more trouble than value.

## How PocketPilot uses it
PocketPilot should help users notice when the environment is too messy, too fragile, or too incomplete to justify pushing ahead as if the answer is obvious.

## Common mistake
A common mistake is treating uncertainty like an insult that must be defeated.
