# SNAPSHOT_VISION.md

## Purpose
Protects Snapshot as PocketPilot’s sacred front-door surface.

## Founding stance
Snapshot is the calm briefing screen, not a mini-dashboard, not an alert graveyard, and not a place to squeeze in one more useful thing.

Its job:
**help the user understand where things stand for their strategy in a few seconds.**

## Non-negotiables
- sacred
- zero-scroll
- instant orientation
- emotional grounding
- quick open-close behavior
- no indicator clutter
- no volatility sirens
- no knowledge-link farm

## Locked core
- Current State
- Last 24h Change
- Strategy Status

## Product rule
Snapshot is a briefing, not a workspace.

## Drift to block
- enrichment drift
- indicator creep
- urgency drift
- profile bloat
- re-entry friction
