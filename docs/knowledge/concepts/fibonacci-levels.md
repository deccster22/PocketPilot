# Fibonacci Levels

## What this means
Common retracement levels traders watch.

## Why it matters
They can help frame potential reaction areas after meaningful moves.

## How to think about it
Treat Fibonacci levels as reference zones, not automatic turning points.

## Common misunderstanding
Assuming price must react at each Fibonacci level.
