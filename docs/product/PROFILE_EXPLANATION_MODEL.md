# PROFILE_EXPLANATION_MODEL.md

## Purpose
Defines how PocketPilot explains the same truth differently across Beginner, Intermediate, and Advanced without turning profiles into different products.

## Founding stance
One Brain, Multiple Voices.

Profiles change:
- density
- explanation depth
- scaffolding visibility
- relationship distance

Profiles do not change:
- event truth
- strategy truth
- confidence truth
- account truth

## Profile intent
### Beginner
Confidence-building translation.

### Intermediate
Contextual narration.

### Advanced
Compressed observation.

## Product rule
Same truth.
Different phrasing.
No contradiction.

## Drift to block
- talking down to beginners
- fake sophistication for advanced users
- profile truth drift
- advice leakage disguised as explanation
