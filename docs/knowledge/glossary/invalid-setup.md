# invalid setup

## Plain-English meaning
An invalid setup is a setup that no longer holds together well enough to deserve trust.

## Canonical definition
In PocketPilot, invalid setup refers to a trade or interpretation idea whose supporting conditions have broken down enough that the original case no longer reads cleanly.

## Why it matters
This helps separate “not perfect” from “no longer good enough.”
