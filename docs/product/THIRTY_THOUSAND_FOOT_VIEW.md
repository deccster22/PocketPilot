# 30,000 ft View

## Product Intent

The 30,000 ft View is PocketPilot's broader stabilising context lane.

It exists for moments when the user may benefit from stepping back briefly and seeing:

- that volatility is elevated or expanding relative to recent conditions
- whether broader structure still feels stable, mixed, or strained
- how Strategy Fit currently reads inside that broader picture
- whether recent interpreted history gives modest grounding to that read

It is not a call to act.
It is not a volatility siren.
It is not a macro theater surface.

## User Experience Rules

- The affordance is quiet and opt-in.
- Snapshot remains primary.
- The detail view reads like a calm debrief, not a command center.
- Fewer stronger points are better than a dense payload.
- Broader context should lower emotional temperature, not raise it.

## Copy Rules

Allowed:

- mixed
- strained
- broader picture
- steadier read
- elevated relative to recent conditions
- expanding relative to recent conditions
- broader structure

Not allowed:

- trade now
- urgent
- warning
- do not miss
- switch strategies
- action recommendations

## Phase Scope

PX-C1 adds:

- one calm Snapshot affordance
- one opt-in detail view
- one canonical service-owned prepared context VM

PX-C2 builds on that by adding:

- one canonical richer interpreted-input seam for volatility, structure, fit support, and light grounding
- somewhat richer detail lines only when the same calm lane already has enough truth to support them
- no new surfaces and no louder delivery behavior

PX-C1 and PX-C2 do not add:

- alerts
- notifications
- background macro polling
- AI commentary
- full regime analytics
