# Candlestick Patterns

## What this means
Patterns showing short-term behaviour.

## Why it matters
They can add compact clues about pressure, hesitation, or rejection.

## How to think about it
Use candle patterns with broader context instead of treating one pattern as decisive.

## Common misunderstanding
Assuming one candle pattern is enough to act.
