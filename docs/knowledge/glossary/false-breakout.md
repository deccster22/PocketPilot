# false breakout

## Plain-English meaning
A false breakout is when price seems to break out, then falls back into the old area instead of holding the move.

## Canonical definition
In PocketPilot, a false breakout is a breakout attempt that fails to build durable acceptance outside the old structure. The move may be meaningful, but it does not become a stable structural shift.

## Why it matters
False breakouts are tricky because the supporting evidence can be partly real. PocketPilot uses them to teach the difference between a breakout attempt and a breakout that actually holds.
