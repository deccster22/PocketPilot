# When a Bounce Is Not a Green Light

## What this means
A bounce can be real without proving the environment is healthy again.

## Why it matters
A [relief rally](../glossary/relief-rally.md) may improve local conditions, but that is not always the same as durable [recovery](../glossary/recovery.md) or a stronger [reversal](../glossary/reversal.md).

## How PocketPilot uses it
PocketPilot should help users separate temporary improvement from deeper repair.

## Common mistake
A common mistake is treating any rebound as proof that the danger has passed.
