## Why This Phase Happened Now

`P7-K19` proved bounded evidence follow-through routing in runtime without adding a visible Examples shelf.  
Accepted `P7-K20` planning then scoped the next safe step: add a small concept follow-through wave so first-wave evidence routes can land on concept pages without `TOPIC_NOT_FOUND`.

## Dependencies

- `P7-K13` for runtime catalog sync validation.
- `P7-K19` for first-wave evidence runtime follow-through baseline.
- Accepted `P7-K20` Concept Runtime Route Planning / Follow-through Readiness scope.

## Why Exactly Five Concepts Were Selected

This wave intentionally includes only:

- `concept-trend`
- `concept-breakout`
- `concept-support`
- `concept-resistance`
- `concept-reversion`

These are the concepts directly needed by the first-wave evidence links from Trend Follow, Breakout Watcher, and Buy the Dip.  
The phase does not surface all concept topics.

## Concept Layer 2 Readiness Merge

Layer 2 readiness payload content was merged into:

- `docs/knowledge/concepts/trend.md`
- `docs/knowledge/concepts/breakout.md`
- `docs/knowledge/concepts/support.md`
- `docs/knowledge/concepts/resistance.md`
- `docs/knowledge/concepts/reversion.md`

Each of these now preserves the beginner Layer 1 top sections and includes a `Layer 2: Deeper understanding` block with:

- `How to recognise this`
- `How it usually develops`
- `What it is commonly confused with`
- `What makes it clearer`
- `What makes it less reliable`
- `How it appears in examples`

Placeholder/future example bridge wording in these five docs was replaced with safe first-wave evidence references.

## Hidden Follow-through Runtime Model

Five concept topic IDs are now runtime-resolvable in `knowledgeCatalog.ts` with:

- `family: core-language`
- `priority: NEXT`

`createKnowledgeLibraryVM` still only shelves `priority: NOW` items, so these concept topics remain hidden follow-through routes rather than visible top-level library cards.

## Evidence-to-Concept Routing Added

First-wave evidence entries now route to bounded concept IDs only:

- trend-follow continuation -> trend, breakout, support
- trend-follow caution -> trend
- breakout acceptance -> breakout, resistance, support

- buy-the-dip continuation -> support, trend

Concept-to-evidence reverse links were intentionally deferred to avoid crowding detail pages in this wave.
